A simple project which scrapes data from trello to aggregate how much people owe for rent.

Assumes you have a list containing cards of checklists for each person who owes rent.

For example:

```
[ List: Rent People Owe ]
    [ Card: July 2015 ]
        [ Checklist: Rent ]
            - Rachel - $500
            - Devnill - $500
            - Kayimbo - $400

        [ Checklist: Utilities ]
            - Rachel - $25
            - Devnill - $25
            - Kayimbo - $50
```
